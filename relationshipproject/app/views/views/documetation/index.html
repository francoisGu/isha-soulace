A)Special Note for Fonts usage - top

Please read this important note before working with this template.

This template uses Font Icomoon to display various icons. It is free! However, we don't have rights to distribute the Font so you need to download the font by yourself.

Follows these steps to add Font Icomoon kit to the template:

  1. Click here to visit Font download site
  2. Unzip the dowloaded file and copy following files to "/font" folder of "Magazine" template:
    1. icomoon.eot
    2. icomoon.svg
    3. icomoon.ttf
    4. icomoon.woff

C) JavaScript - top

This theme imports following Javascript files.

  1. jquery-1.8.3.min.js - Javascript library
  2. jquery.prettyPhoto.js - prettyphoto jQuery plugin
  3. jquery.flexslider.js - Homepage slider
  4. jquery.carouFredSel-6.2.1-packed.js - Related Articles slider
  5. superfish.js - main menu
  6. jquery.validate.min.js - form validator that is used for contact page (contact form) and single page (comment form)
  7. PIE.htc - PIE: CSS3 rendering for IE
  8. PIE_IE678.js - PIE: CSS3 rendering for IE 6, 7 ,8
  9. custom.js - contains custom script

Except custom.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.
